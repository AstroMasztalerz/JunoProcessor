JunoCamProcessor v1.3 README file

===================================================

This is a software package developed to process images from Juno spacecraft JunoCam instrument.
It is developed by @Karol Masztalerz.
It utilises GO-IMG-CONVERTER software by @Paweł Pleskaczyński to support PDS compliant .IMG files
It supports loading of images in both .PNG and .IMG, re-slicing them, creating color composites and basic processing such as sharpening or contrast adjustments
